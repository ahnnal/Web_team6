C Programming Language

The C language was created in 1971 by Dennis Ritchie, who worked at AT&T Bell Laboratory
via Algol 60 based on Fortran I.

C Language's property

  • The C language is a structured language

    Language C provides a way to group commands in a certain unit, and programs made in
    language C are easily maintained because they are clearly divided into units, and they are
    easy to separate the tasks they want from existing programs, so they can be reused when
    developing new programs.

  • The C language is highly portable

    Programs made in the C language work well when taken to other hardware or operating
    systems without modification.The C language was designed so that the C language grammar
    would not be affected even if the hardware changes

  • The C language is highly productive

    Since programmers who speak the C language build their own development infrastructure,
    the more experienced they have, the more diverse the infrastructure becomes, which speeds
    up program development. In addition, the program is more complete and satisfactory
    because it can cope with the various needs of users on its own.

What is the diffrence between C and C++?

C++ is an object-oriented version of the C language.
The difference is that C++ has Class.
